## Task 1: 扩展 format_note_for_response 返回图片列表

<action>
修改 `MediaCrawler/api/routers/notes.py` 中的 `format_note_for_response()` 函数：

1. 在函数中添加以下代码来构建本地图片 URL 列表：
```python
# Build list of all local image URLs
local_image_urls = []
if local_image_count > 0:
    local_image_urls = [f"/images/{note_id}/{i}.jpg" for i in range(local_image_count)]

# Build list of all remote image URLs
image_list_raw = note.get("image_list", "")
if isinstance(image_list_raw, str):
    remote_image_urls = [url.strip() for url in image_list_raw.split(",") if url.strip()]
else:
    remote_image_urls = image_list_raw if image_list_raw else []
```

2. 在返回的字典中添加新字段：
```python
"image_urls": local_image_urls,  # 本地图片URL列表
"remote_image_urls": remote_image_urls,  # 远程图片URL列表
"image_count": max(local_image_count, len(remote_image_urls)),  # 总图片数
```

3. 保留现有的 `local_image_url` 和 `remote_image_url` 字段以保持向后兼容。

</action>

<acceptance_criteria>

- `format_note_for_response()` 函数返回 `image_count` 字段（总图片数）
- 原有字段 `local_image_url`、`remote_image_url`、`first_image_url` 保持不变
