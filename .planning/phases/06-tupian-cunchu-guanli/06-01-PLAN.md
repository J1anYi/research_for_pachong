Create `MediaCrawler/api/services/image_storage.py` with the following implementation:

1. **Imports and configuration:**
   - Import: `hashlib`, `datetime`, `pathlib.Path`, `typing.Optional, Tuple, List`, `PIL.Image`
   - Default DATA_DIR = `Path(__file__).parent.parent.parent / "data"`
   - Default MAX_STORAGE_BYTES = 5 * 1024 * 1024 * 1024 (5GB)

2. **ImageStorageService class:**
   ```python
   class ImageStorageService:
       def __init__(self, data_dir: Path = DATA_DIR, max_storage_bytes: int = MAX_STORAGE_BYTES):
           self._data_dir = data_dir
           self._max_storage_bytes = max_storage_bytes
   ```

3. **Method: get_storage_path(url: str, platform: str, ext: str = '.jpg') -> Tuple[Path, str]**
   - Generate URL hash: `hashlib.md5(url.encode()).hexdigest()[:16]`
   - Create date directory: `{data_dir}/{platform}/images/{year}/{month:02d}/{day:02d}`
   - Create directory with `mkdir(parents=True, exist_ok=True)`
   - Return tuple of (full_path, url_hash)

4. **Method: validate_image(file_path: Path) -> Tuple[bool, Optional[str]]**
   - Try PIL.Image.open() and img.verify()
   - Read first 8 bytes for magic bytes check:
     - `b'\xff\xd8\xff'` = JPEG -> return (True, '.jpg')
     - `b'\x89PNG'` = PNG -> return (True, '.png')
     - `b'GIF8'` = GIF -> return (True, '.gif')
     - `b'RIFF'` (for webp, check bytes 0-4) -> return (True, '.webp')
   - On any exception, return (False, None)
   - If valid but extension mismatch, rename file with correct extension

5. **Method: cleanup_by_size(platform: Optional[str] = None) -> int**
   - Base dir: if platform specified, `{data_dir}/{platform}/images`, else `{data_dir}/*/images`
   - Collect all image files (.jpg, .jpeg, .png, .gif, .webp)
   - Sort by modification time (oldest first)
   - Calculate current total size
   - Delete oldest files until size <= max_storage_bytes
   - Return count of deleted files

6. **Method: get_local_path(url: str, platform: str) -> Optional[Path]**
   - Generate URL hash
   - Search in `{data_dir}/{platform}/images/**/{hash}.*` for existing file
   - Return path if found, else None

7. **Global singleton:** `image_storage = ImageStorageService()`
