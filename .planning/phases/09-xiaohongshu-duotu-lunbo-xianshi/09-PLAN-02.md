## Task 1: 在卡片图片区域添加数量角标

<action>
修改 `MediaCrawler/viewer/static/js/app.js` 中的 `createNoteCard()` 函数：

1. 在 `imageContainer` 创建后，检查图片数量：
```javascript
// 多图数量角标
const imageCount = note.image_count || 1;
if (imageCount > 1) {
    const badge = document.createElement('span');
    badge.className = 'image-count-badge';
    badge.textContent = `${imageCount}图`;
    imageContainer.appendChild(badge);
}
```

2. 在 `MediaCrawler/viewer/static/css/style.css` 中添加角标样式：
```css
.image-count-badge {
    position: absolute;
    right: 8px;
    bottom: 8px;
    background: rgba(0, 0, 0, 0.6);
    color: white;
    font-size: 12px;
    padding: 2px 6px;
    border-radius: 4px;
    font-weight: 500;
}
```

</action>

<acceptance_criteria>
- `app.js` 中 `createNoteCard()` 函数包含多图角标逻辑

- CSS 文件包含 `.image-count-badge` 样式定义
- 角标定位在图片右下角
