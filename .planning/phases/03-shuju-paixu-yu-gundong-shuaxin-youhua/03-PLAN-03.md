#### Task 4: 绑定排序选择器事件
```xml
<read_first>
  - MediaCrawler/viewer/static/js/app.js (setupEventListeners 函数，约 213-245 行)
</read_first>

<action>
在 setupEventListeners() 中添加:
```javascript
// 排序选择
const sortSelect = document.getElementById('sortSelect');
if (sortSelect) {
    sortSelect.value = currentSortOrder;
    sortSelect.addEventListener('change', (e) => {
        currentSortOrder = e.target.value;
        localStorage.setItem(SORT_STORAGE_KEY, currentSortOrder);
        // 重新加载数据
        loadNotes();
    });
}
```
</action>

<acceptance_criteria>
- grep "sortSelect" app.js 返回选择器事件绑定
- grep "localStorage.setItem" app.js 返回保存排序偏好的代码
</acceptance_criteria>
```
