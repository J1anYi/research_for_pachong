## Plan 02: 修复滚动游标刷新无反应问题（实现无限滚动）

### Goal
实现无限滚动加载功能，用户滚动到底部时自动加载更多数据

### Context
- 当前 `setupLazyLoading()` 只处理图片懒加载，不处理列表无限滚动
- API 已支持 `offset` 和 `limit` 分页参数
- 需要实现底部触发器 + 加载逻辑
