## 测试前置条件

1. 服务器运行在 `http://localhost:18080`
2. 小红书数据目录存在数据文件
3. 数据中包含多图笔记

```bash

cd MediaCrawler

# 访问测试页面
# http://localhost:18080/viewer/
```
