# Plan 01: 移除 stats_update 消息的通知触发

## Problem

每个文件变更触发两个 WebSocket 消息：
1. `data_update` → 触发通知
2. `stats_update` → 又触发一次通知

导致提醒框每次文件变更弹出两次。

## Solution

在前端各平台的 WebSocket 处理器中，移除 `stats_update` 类型的通知触发。`stats_update` 仅用于更新统计数据，不触发通知弹窗。

```javascript
} else if (data.type === 'stats_update') {

}
```
</action>
