# 技术栈文档

## 概述

MediaCrawler 是一个多平台社交媒体爬虫项目，支持小红书、抖音、B站、知乎、微博、快手、百度贴吧等平台的数据采集。项目采用 Python 异步架构，结合 Playwright 浏览器自动化和 FastAPI Web 服务，提供了完整的数据采集、存储和可视化解决方案。

## 主要编程语言

### Python
- **版本要求**: Python >= 3.11
- **主要用途**: 后端逻辑、爬虫核心、API服务、数据处理

### JavaScript
- **用途**: 前端可视化界面（纯原生 JS，无框架）
- **文档工具**: VitePress（用于项目文档）

## 构建与部署

### 启动命令
```bash
# 开发服务器
uv run uvicorn api.main:app --port 8080 --reload

# 或直接运行
uv run python -m api.main

# 命令行爬虫
uv run python main.py
```

### 运行时环境
- **ASGI 服务器**: Uvicorn
- **并发模型**: asyncio 异步协程
- **浏览器引擎**: Playwright 内置 Chromium

## 技术架构特点

1. **异步优先**: 全面采用 Python asyncio，实现高并发数据采集
2. **多存储后端**: 支持 JSON/JSONL/CSV/Excel/SQLite/MySQL/PostgreSQL/MongoDB
3. **浏览器自动化**: 使用 Playwright 处理 JavaScript 渲染和反爬虫机制
4. **实时推送**: WebSocket + watchdog 实现数据变更实时通知
5. **代理池支持**: 内置代理 IP 池管理，支持多种代理服务商
