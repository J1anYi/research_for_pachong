## 测试前置条件

1. 服务器运行在 `http://localhost:8080`
2. 安装 yt-dlp: `pip install yt-dlp`
3. B站数据目录存在数据文件

---

## 测试命令

```bash
# 安装 yt-dlp
pip install yt-dlp

# 启动服务器
cd MediaCrawler
uv run uvicorn api.main:app --port 8080 --reload

# 测试 API
curl http://localhost:8080/api/video-tasks/stats
```

---

## 注意事项

1. 视频下载需要网络连接
2. 大视频下载可能需要较长时间
3. 断点续传测试需要手动中断下载
